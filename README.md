# Low-light-Enhanced-Adaptive-Detection-DETR
 designed for detection in underground coal mines’ low-light, high-dust environments. Key innovations include SF-Enhancer (low-light enhancement),Experiments on 11,884-image dataset show mAP@0.5=0.886, 73.65fps, outperforming RT-DETR/YOLO series. Dependencies: Python3.8, PyTorch1.11, CUDA11.8. Clone, install requirements, configure and run.
