# 渐进式去模糊训练实现总结

## 实现概述

成功为RT-DETR实现了**渐进式去模糊训练**，实现了去模糊网络和检测器的协同优化。

### 核心思想

让去模糊网络和RT-DETR互相当老师，循环迭代：
1. **先粗去模糊**：使用简单去模糊网络预处理图像
2. **再RF引导**：利用RT-DETR的多尺度特征作为"RF"引导，优化去模糊
3. **协同优化**：两个网络互相促进，越迭代越清晰

## 已实现的文件

### 1. 核心模块

#### `src/zoo/rtdetr/deblur_module.py`
- **SimpleDeblurNet**: 轻量级去模糊网络（基于ResNet的U-Net架构）
  - 参数量：约290万（轻量版）
  - 支持多种分辨率：128x128 ~ 640x640
  - 残差连接保留细节

- **NAFNetDeblur**: 高质量去模糊网络（基于NAFNet，可选）
  - Simple Gate机制
  - 支持引导图像输入（用于RF引导）

- **DeblurWrapper**: 去模糊网络包装器
  - 支持渐进式训练
  - 可配置的预热迭代次数
  - 自动损失计算（L1/L2/Charbonnier）

#### `src/solver/deblur_solver.py`
- **DeblurDetSolver**: 渐进式去模糊训练求解器

训练阶段：
- **Stage 1 - 预热阶段** (0 ~ warmup_iters)
  - 冻结去模糊网络
  - 训练RT-DETR

- **Stage 2 - 联合训练** (joint_train_start ~ rf_guidance_start)
  - 解冻去模糊网络
  - 同时训练去模糊网络和RT-DETR
  - 去模糊输出 → RT-DETR输入

- **Stage 3 - RF引导** (rf_guidance_start ~ 结束)
  - 使用RT-DETR编码器特征作为引导
  - 特征投影 → 3通道引导图
  - 引导去模糊网络进一步提升质量

#### `src/data/deblur_dataset.py`
- **SyntheticBlur**: 合成模糊变换
  - 高斯模糊（模拟散焦）
  - 运动模糊（模拟运动）
  - 散焦模糊（模拟对焦不准）
  - 可配置模糊概率和类型

- **DeblurDatasetWrapper**: 数据集包装器
  - 支持加载真实模糊-清晰对
  - 在线合成模糊

- **deblur_collate_fn**: 支持模糊-清晰对的collate函数

## 关键参数说明

### 去模糊网络配置

```yaml
deblur_net:
  deblur_net_type: simple      # 'simple' (轻量) or 'nafnet' (高质量)
  num_blocks: [1, 1, 1, 1]     # 每层的残差块数量
  channels: [32, 64, 128, 256] # 特征通道数
  enable_gradual: True         # 渐进式训练
```

**参数量对比**：
- 轻量版（simple, [32,64,128,256]）：约290万参数
- 高质量版（simple, [64,128,256,512]）：约1100万参数

### 训练阶段配置

```yaml
warmup_iters: 1000           # 预热阶段迭代数
joint_train_start: 2000      # 开始联合训练的迭代数
rf_guidance_start: 5000      # 开始RF引导的迭代数
use_rf_guidance: True         # 是否启用RF引导
deblur_loss_weight: 0.5      # 去模糊损失权重
```

**调整建议**：

| 场景 | warmup_iters | joint_train_start | rf_guidance_start | deblur_loss_weight |
|------|--------------|-------------------|-------------------|-------------------|
| 小数据集（<1K图） | 500 | 1000 | 2000 | 0.3 |
| 中等数据集（1K-10K） | 1000 | 2000 | 5000 | 0.5 |
| 大数据集（>10K） | 2000 | 5000 | 10000 | 0.7 |

### 数据增强配置

```yaml
train_dataloader:
  dataset:
    transforms:
      ops:
        - {type: SyntheticBlur,
           blur_prob: 0.5,                    # 模糊概率
           blur_types: ['gaussian', 'motion']}  # 模糊类型
```

## 性能优化

### 训练速度优化

1. **使用轻量级网络**
```yaml
num_blocks: [1, 1, 1, 1]
channels: [32, 64, 128, 256]
```

2. **减小batch size + 梯度累积**
```yaml
batch_size: 2
optimizer: {accum_iter: 4}  # 等效batch_size=8
```

3. **使用混合精度训练**
```bash
python tools/train.py -c config.yml --amp
```

```python
# 在模型定义中使用
from torch.utils.checkpoint import checkpoint
```

2. **减小输入分辨率**
```yaml
- {type: Resize, size: [512, 512]}  # 从640降到512
```

## 与现有功能的集成

### 与HEFF模块的协同

`rtdetr_r50vd_meikuang_deblur.yml` 配置示例展示了如何将去模糊训练与HEFF（Encoder-Free）模块结合使用：

```yaml
HybridEncoder:
  encoder_free: True
  heff_enable: True
  heff:
    num_fusion_layers: 1
    block_type: basic
    share_weights: True
```

### 与去噪训练的协同

去模糊和去噪可以同时训练：
- Stage 1-2: 主要训练去模糊
- Stage 3 (RF引导): 同时改善去模糊和去噪效果
